Typically, Node.js web application libraries will be located in a `yarn` file or `package.json`

### When it comes to Node.js web applications, the following is a list of security-specific libraries:

Input Sanitization/Validation
===

- Joi
- Validator
- Express-Validator

Authentication
===

- jsonwebtoken
- passport.js
- express-jwt

Authorization
===

- casbin
- accesscontrol

Secure Session Management
===

- express-session
- cookie-session
- cookie-parser

Web Security Headers
===

- helmet
- csurf
- hsts
- cors

Cryptograph
===

- bcrypt
- crypto
- argon2

Secure HTTP
===

- express-rate-limit
- hpp

Logging
===

- Winston
- bunyan

Misc
===

- rate-limiter-flexible
- express-secure-only
- DOMPurify
